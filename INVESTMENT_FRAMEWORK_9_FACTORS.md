# 📊 Decision Stream 투자 판단 프레임워크

## 철학: 가치와 모멘텀을 넘어서

**"주가는 가치가 아니라 돈이 움직이는 방향으로 간다"**

Decision Stream은 단순한 밸류에이션이나 모멘텀을 넘어, 시장의 입체적 구조를 읽는 9가지 요소 프레임워크를 제공합니다.

---

## 🎯 9가지 핵심 요소

### 1️⃣ 자본의 흐름 (Capital Flow) ⭐ 최우선

> "돈이 들어오지 않는 가치주는 오래 눌립니다"

**Decision Stream 구현:**
- **Sector Heatmap**: SURGE 신호로 자금 집중 섹터 자동 감지
- **Flow Score**: 거래대금, 기관/외국인 순매수, ETF 유입 종합
- **Duration**: 자금 흐름 지속 기간 추적

**체크 포인트:**
- [ ] 기관·외국인 수급 변화
- [ ] ETF 자금 유입/유출
- [ ] 테마 단위 자금 이동 (AI → 반도체 → 전력)

**강한 의견:**
개별 기업 분석보다 **어느 시장·섹터로 돈이 들어오는지**를 먼저 봐야 합니다.

---

### 2️⃣ 사이클 인식 (Cycle)

> "기업은 혼자 움직이지 않습니다"

**Decision Stream 구현:**
- **Market Regime**: 경기 국면 자동 판단 (Risk-On/Off)
- **Sector Stage**: 산업 사이클 단계 (초기/성장/성숙/쇠퇴)
- **Policy Driver**: 정책 변화 추적 (금리, 재정, 규제)

**체크 포인트:**
- [ ] 경기 국면: 확장 / 둔화 / 침체 / 회복
- [ ] 산업 사이클: 어느 단계인가?
- [ ] 정책 사이클: 금리, 재정, 지정학

**예시:**
- 금리 인상기 → 현금흐름 좋은 기업
- 정책 드라이브 초기 → 기대주가 먼저 움직임

---

### 3️⃣ 기업의 질 (Quality)

> "불황이 오면 나쁜 기업은 50% 하락, 좋은 기업은 30% 조정으로 끝납니다"

**Decision Stream 구현:**
- **Quality Score**: 부채 구조, 현금흐름, 영업이익 안정성
- **Resilience Index**: 위기 생존 가능성
- **Pricing Power**: 원가 전가력 (가격 결정권)

**체크 포인트:**
- [ ] 부채 구조 (만기, 고정/변동 금리)
- [ ] 현금흐름의 안정성
- [ ] 영업이익의 반복 가능성
- [ ] 원가 전가력

**강한 의견:**
회복 속도가 수익을 가릅니다.

---

### 4️⃣ 지배구조 (Governance)

> "회사는 돈을 벌지만 주주는 못 번다 - 이 상황을 피하는 것이 수익의 절반"

**Decision Stream 구현:**
- **Governance Score**: 주주환원 정책, 자사주 매입
- **Ownership Structure**: 대주주 지분, 경영진 보유
- **Track Record**: 과거 위기 대응 기록

**체크 포인트:**
- [ ] 자사주 매입/소각 여부
- [ ] 주주환원 정책의 일관성
- [ ] 대주주 지분 구조
- [ ] 과거 위기 대응 기록

**한국 시장의 현실:**
같은 산업, 같은 실적이라도 주가 결과는 전혀 다릅니다.

---

### 5️⃣ 서사 (Narrative)

> "아무리 좋은 숫자도 이야기가 없으면 리레이팅은 없습니다"

**Decision Stream 구현:**
- **Narrative Score**: 시장 이해도, 글로벌 담론 연결성
- **Story Strength**: 미래 가능성, 시장의 믿음
- **Theme Alignment**: AI, 에너지, 고령화, 국방 등

**체크 포인트:**
- [ ] 이 기업이 미래에 무엇이 될 수 있는가?
- [ ] 시장이 이해하기 쉬운가?
- [ ] 글로벌 담론과 연결되는가?

**냉정한 현실:**
주가 = 팩트 + 이야기

---

### 6️⃣ 하방 리스크 (Downside Risk)

> "이 종목이 틀렸을 때, 나는 살아남는가?"

**Decision Stream 구현:**
- **Risk Score**: 하방 제한 구조, 악재 반영도
- **Asymmetric Return**: Upside vs Downside 비율
- **Exposure Check**: 레버리지, 환율, 원자재

**체크 포인트:**
- [ ] 하방이 제한된 구조인가?
- [ ] 악재가 이미 가격에 반영되었는가?
- [ ] 레버리지/환율/원자재 노출도

**노련한 투자자의 질문:**
"얼마나 오를 수 있나"보다 **"최악의 경우 얼마나 잃을 수 있나"**가 먼저입니다.

---

### 7️⃣ 시간 적합성 (Time Fit)

> "자기 자금의 시간표에 맞는 투자가 중요합니다"

**Decision Stream 구현:**
- **Time Horizon Match**: 투자 기간 vs 기업 성장 시간
- **Opportunity Cost**: 기회비용 계산
- **Capital Rotation**: 자금 회전 속도

**체크 포인트:**
- [ ] 기업의 성장 시간표
- [ ] 나의 자금 회전 속도
- [ ] 기회비용

**겸손한 조언:**
좋은 기업 ≠ 지금 좋은 투자

---

### 8️⃣ 가치 (Valuation)

> "기본이지만 충분하지 않다"

**Decision Stream 구현:**
- **Value Score**: PER, PBR, PSR, EV/EBITDA
- **Relative Valuation**: 동종 업계 대비
- **Historical Range**: 과거 밸류에이션 밴드

**체크 포인트:**
- [ ] 절대 밸류에이션
- [ ] 상대 밸류에이션
- [ ] 역사적 범위

---

### 9️⃣ 모멘텀 (Momentum)

> "가치 있어도 모멘텀 없으면 기다려야 합니다"

**Decision Stream 구현:**
- **Momentum Score**: 가격 추세, RSI, MACD
- **Structure**: Higher High, Higher Low
- **Volume**: 거래량 패턴

**체크 포인트:**
- [ ] 가격 추세
- [ ] 기술적 구조
- [ ] 거래량 확인

---

## 🎯 투자 판단 체크리스트

Decision Stream은 9가지 요소를 자동으로 점수화하여 제공합니다:

### 필수 체크 (모두 통과해야 매수)
1. ✅ **자금 흐름** (Flow Score ≥ 70)
2. ✅ **사이클** (적절한 국면)
3. ✅ **기업의 질** (Quality Score ≥ 60)
4. ✅ **지배구조** (Governance Score ≥ 50)
5. ✅ **서사** (Narrative 존재)
6. ✅ **하방 리스크** (Risk Score ≤ 30)
7. ✅ **시간 적합성** (Time Fit = True)

### 보조 체크 (가산점)
8. ✅ **가치** (Valuation 저평가)
9. ✅ **모멘텀** (Momentum 상승)

---

## 📊 Decision Stream의 9요소 통합

### Stock Funnel 분류 로직

```python
def classify_stock(stock):
    """
    9가지 요소 종합 판단
    """
    # 1. 자금 흐름 (최우선)
    flow_score = calculate_flow_score(stock)
    
    # 2. 사이클
    cycle_fit = check_cycle_fit(stock)
    
    # 3. 기업의 질
    quality_score = calculate_quality(stock)
    
    # 4. 지배구조
    governance_score = calculate_governance(stock)
    
    # 5. 서사
    narrative_score = calculate_narrative(stock)
    
    # 6. 하방 리스크
    risk_score = calculate_downside_risk(stock)
    
    # 7. 시간 적합성
    time_fit = check_time_fit(stock)
    
    # 8. 가치
    value_score = calculate_valuation(stock)
    
    # 9. 모멘텀
    momentum_score = calculate_momentum(stock)
    
    # 종합 판단
    if flow_score >= 70 and cycle_fit and quality_score >= 60 \
       and governance_score >= 50 and narrative_score >= 60 \
       and risk_score <= 30 and time_fit:
        if momentum_score >= 70:
            return "LEADER"      # 지금 매수
        else:
            return "FOLLOWER"    # 눌림 매수 대기
    else:
        return "NO_GO"           # 회피
```

## 🎬 실전 적용 예시

### Case 1: 방산 섹터 - LMT (Lockheed Martin)

| 요소 | 점수 | 판단 |
|------|------|------|
| 1. 자금 흐름 | 97 | ✅ SURGE |
| 2. 사이클 | ✅ | 지정학 긴장 |
| 3. 기업의 질 | 85 | ✅ 현금흐름 우수 |
| 4. 지배구조 | 80 | ✅ 주주환원 우수 |
| 5. 서사 | 90 | ✅ 국방 담론 강함 |
| 6. 하방 리스크 | 20 | ✅ 제한적 |
| 7. 시간 적합성 | ✅ | 중기 스윙 적합 |
| 8. 가치 | 70 | ✅ 적정 |
| 9. 모멘텀 | 62 | ⚠️ 아직 돌파 전 |

**결론: FOLLOWER (눌림 매수 대기)**
- 자금은 들어오지만 아직 기관 돌파 전
- 조정 시 매집 기회

### Case 2: 헬스케어 - JNJ (Johnson & Johnson)

| 요소 | 점수 | 판단 |
|------|------|------|

| 2. 사이클 | ✅ | 방어 수요 |
| 3. 기업의 질 | 90 | ✅ 최상급 |
| 4. 지배구조 | 95 | ✅ 배당 귀족주 |
| 5. 서사 | 85 | ✅ 고령화 |
| 6. 하방 리스크 | 10 | ✅ 매우 낮음 |
| 7. 시간 적합성 | ✅ | 중장기 적합 |

**결론: LEADER (추세 추종)**
- 안정형 투자자 최적
