## 💡 핵심 메시지 (10가지)

1. **자금 흐름이 최우선** - 돈이 들어오지 않는 가치주는 오래 눌립니다
2. **사이클을 읽어라** - 기업은 혼자 움직이지 않습니다
3. **질이 회복 속도를 결정** - 불황 시 생존율이 다릅니다
4. **지배구조가 수익의 절반** - 회사 실적 ≠ 주주 수익
5. **서사가 없으면 리레이팅 없음** - 주가 = 팩트 + 이야기
6. **하방부터 체크** - 살아남는 것이 우선입니다
7. **시간표가 맞아야** - 좋은 기업 ≠ 지금 좋은 투자
8. **가치는 기본** - 하지만 충분하지 않습니다
9. **혼자 오르면 위험, 같이 오르면 돈 냄새** ⭐ - 모멘텀 품질이 결정적
10. **계좌를 지키며 반복하는 것** ⭐ - 승률보다 Risk/Reward

### Case 1: 방산 섹터 - LMT (Lockheed Martin)

#### 9요소 분석
```
1. 자금 흐름: 97점 ✅ (SURGE)
2. 사이클: ✅ (지정학 긴장)
3. 기업의 질: 85점 ✅
4. 지배구조: 80점 ✅
5. 서사: 90점 ✅ (국방 담론)
6. 하방 리스크: 20점 ✅
7. 시간 적합성: ✅ (중기)
8. 가치: 70점 ✅
9. 모멘텀 품질: 85점 ✅ (진짜)
```

#### 모멘텀 품질 상세
```
✅ 섹터 동반 상승: LMT, 012450, 079550 동시 상승
✅ 기관+외국인: 동시 순매수
✅ 거래대금: 3일 연속 증가
✅ 뉴스 성격: "국방비 증액 확정" (정책)
⚠️ 커뮤니티: 약간 과열
```

#### 실전 체크리스트
```
✅ 1. 돈이 들어오고 있는가? YES
✅ 2. 같이 오르는가? YES (3개 종목)
✅ 3. 재료 수명 충분? YES (정책 확정)
✅ 4. 가격 구조 건강? YES
✅ 5. 시장 우호적? YES
✅ 6. 늦은 테마? NO
✅ 7. 손절 명확? YES
✅ 8. R/R ≥ 2? YES
✅ 9. 규칙 매매? YES
✅ 10. 살아남는 구조? YES

결과: 10/10 통과 → FOLLOWER (눌림 매수 대기)
```

#### 최종 판단
```
분류: FOLLOWER
행동: BUY_PULLBACK (눌림 매수 대기)
이유: 섹터 강함, 진짜 모멘텀, 종목은 돌파 전
신뢰도: HIGH
```

### Case 2: 가짜 모멘텀 종목 (예시)

#### 모멘텀 품질 상세
```
❌ 혼자 오름: 섹터 다른 종목 무반응
❌ 기관+외국인: 매도 중
❌ 거래대금: 1일만 폭발
❌ 뉴스 성격: "~할 전망" (루머)
❌ 커뮤니티: 과열 (유튜버 동시 언급)
❌ 과거 패턴: 급등급락 이력

결과: 25점 → 가짜 모멘텀
```

#### 최종 판단
```
분류: NO_GO
행동: AVOID (절대 회피)
이유: 가짜 모멘텀 감지
경고: 혼자 오르는 종목은 위험
```
