# 📊 Market Regime & Stock Funnel 해석 가이드

## 🎯 Decision Stream 시장 분석 프레임워크

이 문서는 Decision Stream Dashboard의 Market Regime과 Stock Funnel을 올바르게 해석하고, 실전 매매 전략으로 연결하는 방법을 설명합니다.

---

## 📈 Market Regime 해석

### 1. RISK_ON vs RISK_OFF

#### RISK_ON (위험자산 선호)
```
조건:
- Risk Score ≥ 70
- 변동성 안정
- 대형주 유입
- 외국인/기관 순매수

의미:
→ 자금이 주식 시장으로 유입되는 국면
→ 공격적 포지션 가능
→ 섹터 선도주 발굴 최적 타이밍
```

#### RISK_OFF (안전자산 선호)
```
조건:
- Risk Score < 50
- 변동성 급증
- 소형주 이탈
- 외국인/기관 순매도

의미:
→ 자금이 주식 시장에서 빠져나가는 국면
→ 신규 진입 금지
→ 기존 포지션 축소 또는 현금 확보
```

---

### 2. Playbook 해석

#### 눌림 매수 (Pullback Entry)
```
환경: RISK_ON + 변동성 안정
전략: 추격 금지, 조정 후 재진입

실행 조건:
1. 섹터가 SURGE 신호
2. 종목이 Leader 또는 Follower
3. MA20 터치 후 반등 시작
4. 거래량 증가 확인

승률: 가장 높음 (60-70%)
```

#### 공격적 진입 (Aggressive Entry)
```
환경: RISK_ON + Risk Score > 80
전략: 돌파 매수, 포지션 확대

실행 조건:
1. 섹터 SURGE + Leader 돌파
2. 거래량 급증
3. 뉴스 확정 + 지속성

주의: 손절가 엄수 필수 (-10%)
```

#### 관망 (Wait & See)
```
환경: RISK_OFF 또는 Risk Score < 50
전략: 신규 진입 금지, 현금 확보

실행:
- 기존 포지션 정리 (20일선 이탈 시)
- 현금 비중 70% 이상 유지
- 다음 RISK_ON 신호 대기
```

## 🌡️ Sector Heatmap 해석

### 1. SURGE 신호 의미

**SURGE ≠ 단순 강세**

SURGE는:
- **현재 이 순간 자금이 몰리는 곳**
- 수급(외국인/기관) + 가격(상승) + 거래량(증가) 동시 충족
- 단기(1-2주) 또는 중기(1-3개월) 지속 가능

#### 예시: 방산 섹터 SURGE (Score 97)
```
의미:
→ 방산 섹터로 대규모 자금 유입 중
→ 정책·예산·계약 뉴스 확정
→ 섹터 내 종목 선별 매수 적기

실전 전략:
1. Stock Funnel에서 Leader/Follower 확인
2. 눌림 구간 진입 (MA20 기준)
3. 포지션 크기: 25-30% (공격적 기준)
```

---

### 2. NORMAL 신호 의미

**NORMAL = 자금 유입/유출 없음**

NORMAL은:
- 현재 자금이 정체 또는 이탈 중
- 강세 후 쉬는 구간 (조정)
- 또는 아직 시작 전 (대기)

#### 예시: AI 반도체 NORMAL (Score 46)
```
의미:
→ AI 반도체는 나쁘지 않지만 지금은 돈이 빠지거나 쉬는 구간
→ 이전에 강했으나 현재는 자금 유출 중

실전 전략:
1. 신규 진입 보류
2. 기존 보유 종목은 20일선 기준 관리
3. 다음 SURGE 신호 대기
```

---

### 3. 점수별 행동 지침

| 점수 | 신호 | 행동 |
|------|------|------|
| 90-100 | SURGE (강) | 적극 진입, 포지션 30% |
| 70-89 | SURGE (중) | 선별 진입, 포지션 20-25% |
| 50-69 | NORMAL | 관망, 신규 진입 보류 |
| 0-49 | NORMAL (약) | 진입 금지, 기존 정리 검토 |

## 🎯 Stock Funnel 해석

### 1. Leader (선도주)

**Leader = 섹터를 이끄는 종목**

특징:
- 섹터 내 가장 먼저 상승
- 가장 강한 수급 (외국인+기관 동시 매수)
- 거래대금 1-3위
- 뉴스 직접 수혜

#### Leader 있을 때
```
예시: 헬스케어 → JNJ (Leader)

의미:
→ JNJ가 헬스케어 섹터의 절대적 선도주
→ 나머지는 아직 못 따라오고 있음

실전 전략:
- JNJ 추세 추종 (안정적)
- 돌파 시 추가 매수 가능
- 포지션 크기: 20-25%
```

#### Leader 없을 때
```
예시: 방산 → Leader 없음, Follower만 존재

의미:
→ 섹터는 강하지만 명확한 선도주가 아직 없음
→ 종목들이 본격 돌파 전 단계

실전 전략:
→ 가장 좋은 매수 기회 직전!
→ Follower 중 가장 강한 종목 눌림 매수 대기
→ 포지션 크기: 20% (신중)
```

---

### 2. Follower (추종주)

**Follower = Leader를 따라가는 종목**

특징:
- Leader보다 늦게 상승 시작
- 수급은 양호하나 Leader보다 약함
- 기술적 지지선 확인 필요
- 2차 상승 가능성

#### 전략
```
예시: 방산 → LMT, 한화에어로 (Follower)

의미:
→ 섹터가 먼저 움직임
→ 종목들은 아직 본격 돌파 안 함

실전 전략:
1. MA20 눌림 매수 대기
2. 거래량 증가 확인 후 진입
3. 1차 목표: +15-20% (Leader보다 낮게 설정)
4. 포지션 크기: 20-25%
```

---

### 3. No-Go (진입 금지)

**No-Go = 리스크 높은 종목**

특징:
- 과열 후 조정
- 수급 악화 (외국인+기관 매도)
- 뉴스 과열 또는 실적 우려
- 기술적 지지선 이탈

#### 전략
```
No-Go 종목은 어떤 경우에도 진입 금지!

이유:
→ 손실 확률 70% 이상
→ 단기 과열 후 조정 국면
→ 추가 하락 가능성 높음

대신:
- 조정 완료 후 Follower로 승격 대기
- 최소 3-6개월 모니터링
```

## 💡 실전 사례 분석

### 사례 1: RISK_ON + 방산 SURGE + Follower

**시장 상황:**
```
Market Regime:
- State: RISK_ON
- Risk Score: 72
- Playbook: 눌림 매수
- 변동성: 안정
- 자금 흐름: 대형주 유입

Sector Heatmap:
- 방산: 97 (SURGE)

Stock Funnel:
- Leader: 없음
- Follower: LMT, 한화에어로 (012450)
- No-Go: 없음
```

**해석:**
```
이 조합은 "공격 자금이 방산으로 이동 중인 Risk-On 장"

가장 좋은 매수 기회 직전 구조!
```

**실전 전략:**
```
1단계: 종목 선택
   → 한화에어로 (012450) 선택
   → 국내 대표 방산주 + 수급 양호

2단계: 진입 타이밍
   → MA20 눌림 대기 (현재가의 3-5% 하락)
   → 거래량 증가 확인 후 진입

3단계: 포지션 설정
   - 투자 금액: 5,000만 원
   - 포지션 크기: 25% (1,250만 원)
   - 리스크 성향: 중립

4단계: 손절/익절 설정
   - 진입가: 85,000원 (눌림 대기)
   - 손절가: 76,500원 (-10%)
   - 목표가 1차: 102,000원 (+20%)
   - 목표가 2차: 114,750원 (+35%)

5단계: 포지션 관리
   - 1차 익절: 50% 매도
   - 2차 익절: 30% 매도
   - 잔량: 20일선 이탈 시 정리
```

**기대 결과:**
```
승률: 65%
평균 수익률: +12%
손익비: 2.4:1
```

### 사례 2: RISK_ON + 헬스케어 SURGE + Leader

**시장 상황:**
```
Market Regime:
- State: RISK_ON
- Risk Score: 72

Sector Heatmap:
- 헬스케어: 96 (SURGE)

Stock Funnel:
- Leader: JNJ (존슨앤존슨)
- Follower: 없음
```

**해석:**
```
"한 종목에 자금이 집중되고,
아직 섹터 확산은 안 된 단계"

→ 안정형, 방어형 트레이드에 최적
```

**실전 전략:**
```
1단계: 종목 선택
   → JNJ (절대적 선도주)

2단계: 진입 방식
   → 추세 추종 (Trend Following)
   → 눌림보다는 돌파 우선

3단계: 포지션 설정
   - 투자 금액: 5,000만 원
   - 포지션 크기: 20% (1,000만 원) ← 안정형
   - 리스크 성향: 보수

4단계: 손절/익절 설정
   - 진입가: 현재가
   - 손절가: -8%
   - 목표가 1차: +15%
   - 목표가 2차: +25%
```

**기대 결과:**
```
승률: 70%
평균 수익률: +10%
손익비: 2.0:1
리스크: 낮음 (대형주 + 안정 섹터)
```

## 🎓 전략 선택 가이드

### 투자 성향별 최적 전략

#### 공격적 투자자
```
환경: RISK_ON + SURGE 섹터
목표: Follower 눌림 매수

이유:
- Follower는 Leader보다 변동성 크지만 수익률 높음
- 눌림 매수로 진입가 최적화
- 포지션 25-30%

적합 사례:
- 방산 섹터 (SURGE) → LMT, 한화에어로 (Follower)
```

#### 안정적 투자자
```
환경: RISK_ON + Leader 존재
목표: Leader 추세 추종

이유:
- Leader는 변동성 낮고 승률 높음
- 추세 추종으로 안정적 수익
- 포지션 20%

적합 사례:
- 헬스케어 섹터 (SURGE) → JNJ (Leader)
```

#### 현금 많은 투자자
```
환경: RISK_ON + 복수 SURGE 섹터
목표: 분산 투자 (2-3종목)

전략:
1. 방산 → Follower 1종목 (20%)
2. 헬스케어 → Leader 1종목 (20%)
3. 현금 비중 60% 유지

목표: 안정적 8-12% 연 수익
```

## ⚠️ 실전 주의사항

### 1. 추격 매수 금지 원칙

**추격 매수 = 뉴스 당일 또는 급등 후 매수**

```
절대 금지 상황:
- 뉴스 당일 +10% 이상 급등
- SURGE 신호 나온 지 1주일 이내
- Leader가 이미 +30% 이상 상승
```

**대신:**
```
✓ 눌림 대기 (MA20 터치)
✓ 거래량 감소 후 재증가 확인
✓ 조정 완료 후 진입
```

---

### 2. No-Go 종목 절대 회피

**No-Go는 어떤 이유로도 진입 금지!**

```
No-Go 판정 이유:
- 과열 후 조정 (10일 +30% 이상)
- 수급 악화 (외국인+기관 매도 전환)
- 뉴스 과열 (실적 지연 우려)
- 기술적 지지선 이탈
```

**통계:**
```
No-Go 진입 시:
- 손실 확률: 70%
- 평균 손실: -15%
- 회복 기간: 3-6개월
```

---

### 3. RISK_OFF 시 대응

**RISK_OFF 신호 발생 시:**

```
즉시 실행:
1. 신규 진입 전면 중단
2. 기존 포지션 20일선 기준 관리
3. 20일선 이탈 시 즉시 정리
4. 현금 비중 70% 이상 확보
5. 다음 RISK_ON 신호 대기
```
