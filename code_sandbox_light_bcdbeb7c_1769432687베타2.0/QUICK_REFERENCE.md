## 📊 7요소 체크리스트 (총 100점)

| 항목 | 배점 | 핵심 기준 |
|------|------|-----------|
| 수급 신호 | 25점 | 외국인/기관 5일+ 매수 |
| 정책/테마 | 30점 | 확정 + 지속 + 실적연결 |
| 시장 사이클 | 10점 | MA20 위 + 낮은 변동성 |
| 기업 질 | 10점 | 부채 < 100%, 수익성 양호 |
| 서사 | 8점 | 산업 성장 + 업계 리더 |
| 하방 리스크 | 10점 | 과열 無, MA20 지지 |
| 시간 적합성 | 7점 | 조정 후 진입 타이밍 |

**✅ 70점 이상 → 진입 고려**
**⚠️ 55-70점 → 포지션 축소**
**🚫 55점 미만 → 관망 권장**

## 📈 성과 분석 지표

### 추적 지표
- **승률**: 익절 종료 / 전체 시뮬레이션
- **평균 수익률**: 총 수익률 / 전체 건수
- **손익비**: 평균 익절 / 평균 손절
- **최대 연속 손실**: 연속 손절 횟수

### 개선 방법

**승률 낮을 때 (<40%)**:
- 체크리스트 통과 기준 강화 (70→75점)
- RISK_OFF 시 진입 금지
- 조정 후 진입 원칙 준수

**평균 수익률 낮을 때 (<3%)**:
- 익절 목표가 재조정
- 포지션 크기 확대 검토
- 고점수 종목만 선택 (75점+)

**손익비 낮을 때 (<1.5:1)**:
- 손절 타이밍 점검
- 2차 익절까지 보유
- 수급·정책 항목 강화

## 📞 자주 묻는 질문 (FAQ)

**Q: 시뮬레이션 기록이 사라졌어요!**
A: 같은 브라우저의 같은 도메인에서 접속하세요. 시크릿 모드는 기록이 유지되지 않습니다.

**Q: 실제 매매는 어떻게 하나요?**
A: 이 도구는 시뮬레이션 전용입니다. 실제 매매는 증권사 HTS/MTS를 사용하세요.

**Q: 7요소 점수가 자동으로 계산되나요?**
A: 현재는 목업 데이터로 작동합니다. 실제 점수는 백엔드 API 연동 후 정확해집니다.
